Introduction

HTML, or Hypertext Markup Language, is the foundation of web development. It provides the structure and content of web pages, allowing you to create documents that can be viewed in web browsers.

HTML is a versatile language that can be used on its own to create static web pages or combined with other technologies like CSS (Cascading Style Sheets) and JavaScript to build interactive and dynamic web applications.

This is the general syntax of a Element in Html:

<element attribute="attribute_value">Content</element>

For Example:

<form action="#"></form>
Tags

HTML elements are defined using tags, which are enclosed in angle brackets < >. Tags usually come in pairs: an opening tag and a closing tag. The opening tag specifies the beginning of an element, and the closing tag marks the end of the element. For example, <p> is the opening tag for a paragraph element, and </p> is its closing tag.</p>

Some popular tag:

<div></div>
<p></p>
<a />
<img />
<ul></ul> <ol></ol>
<li></li>
<h1></h1> <h6></h6>
<span></span>
<input />
<form></form>
  • The div tag: Generic container used for grouping and styling content together, commonly for layout and CSS styles.
  • The p tag: Defines a paragraph of text within a document.
  • The a tag: Creates a hyperlink to another webpage or resource.
  • The img tag: Inserts an image into a webpage with alternative text for accessibility.
  • The ul tag and the ol tag: Create unordered and ordered lists, respectively.
  • The li tag: Defines a list item within an unordered or ordered list.
  • The h1 tag to the h6 tag: Define headings of different levels.
  • The span tag: Inline container for applying styles to specific text or grouping inline elements.
  • The input tag: Creates a text input field within a form.
  • The form tag: Creates an HTML form for collecting user input.
Attributes

Attributes in HTML provide additional information about an element and are always specified in the start tag of an element.Attributes can define characteristics such as the appearance, behavior, or functionality of an element.

For Example:

<img src="" />

The src attribute of the img tag specifies the source URL (Uniform Resource Locator) of the image to be displayed on a web page. This attribute is mandatory for the img tag, as it tells the browser where to find the image file that should be rendered.

Content

Content includes text, images, videos, links, and any other elements or data that are enclosed within an HTML tag. In simple terms, it's what you see on the webpage when you view it in a web browser.

For Example:

<p> This is CONTENT </p>
